# Claude Skills - Recomendaciones de Negocios WhatsApp

Colección profesional de Claude Skills para un sistema de recomendaciones de negocios integrado con WhatsApp.

## Estructura

```
.claude/skills/
├── business-analyzer/        # Análisis y clasificación de negocios
├── recommendation-engine/    # Motor de recomendaciones
├── whatsapp-integration/     # Integración con WhatsApp
├── data-processing/          # Procesamiento de datos
└── utilities/                # Funciones auxiliares
```

## Skills Disponibles

### 1. Business Analyzer
Analiza características de negocios y categoriza según criterios específicos.

### 2. Recommendation Engine
Genera recomendaciones personalizadas basadas en preferencias del usuario.

### 3. WhatsApp Integration
Formatea y envía mensajes a través de WhatsApp.

### 4. Data Processing
Procesa y normaliza datos de entrada.

### 5. Utilities
Funciones de soporte y herramientas comunes.

## Uso

Cada skill se puede invocar independientemente o como parte de un flujo integrado.
